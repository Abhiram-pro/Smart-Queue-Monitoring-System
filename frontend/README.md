# Frontend - Smart Queue Monitoring System

React TypeScript dashboard for real-time queue monitoring.

## Setup

```bash
npm install
```

## Development

```bash
npm start
```

## Build

```bash
npm run build
```

## Environment Variables

Create `.env` file:

```
REACT_APP_API_URL=http://localhost:5000
REACT_APP_WS_URL=ws://localhost:8765
```

## Deploy to Vercel

```bash
vercel --prod
```

Set environment variables in Vercel dashboard after deployment.
